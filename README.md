# QA Manual Testing Portfolio (FR / EN)

This repository contains a structured portfolio of **manual QA testing artifacts**.

The goal is to demonstrate the ability to:
- define a clear test scope,
- write structured QA documentation,
- design test cases and bug reports for a web application.

---

## Scope
- Manual web testing
- QA documentation
- No automation
- Intentionally limited functional scope
